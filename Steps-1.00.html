How to Load the Data?

Download the PBIX data file provided. All the necessary tables will all be within this PBIX.file including the tables and measures, and filters involved in making the scatter plot. Once you download and up open the file using Power BI desktop it should look like the image below.


In order to load the PBIX file, make sure to first download Power BI Desktop. It is free to download.

https://powerbi.microsoft.com/en-us/desktop/
