How to add the conditional coloring of plot points?

Now we are going to add some additional features to make the plot points in the data stand out more and be more visually informative. We will incorporate a condition where the plot points that are above the threshold of both axis will be colored green, plot points that only go past the axis of one of the axes will be colored yellow and plots that do not meet both thresholds will be in red. So we will have something that looks like the image below.  The image indicates what each quadrants color would be. 

The first order of business is to open the format tab of the visualizations menu which is indicated with the red circle and scroll down to Data Colors and expand it with the arrow. Then click the advanced controls link which will take you to a separate window. Refer to the image below with the text highlighted in green.

The window will look something like the image below.

Go to the Format by menu and select Rules. Then in the Based on field, click the drop-down menu and select COLORQUADRANT from the Key Measures area. The image below shows where the drop-down window is located within the menu. 

Now we can proceed to work on the rules. Enter the following values shown in the image below. To add a new rule simply click on the yellow box named +Add to add the two other rules. Also, customize the colors for each selection to get the brightest colors of yellow, green and red. Refer to the image below which will indicate what to enter in each rule. 


You can customize each color by first clicking the downwards arrow of the color box and then clicking custom color shown in the picture below. Then select the brightest version of each of the colors in the second menu. The two images below show how to customize the colors.

 

After creating the three rules, selecting the colors and clicking the OK button. The graph should look something like this shown in the image below.
