How to create the scatter plot?

Once you have loaded and opened the file, start by clicking on the scatter plot icon indicated in the image below. The scatter plot icon will then create a chart that will pop up on the report.


Drag down the bottom of the scatter plot so that the chart fills the entire space we are working with shown like the image below.



Now we can begin with the actual data. First click on the scatter plot image and options to edit the Details, Legend, X and Y axis and so forth will show up under the visualizations menu. This tab is called fields, which is indicated by the red circle, and is where different columns and measures will be added to create the plot points of the scatter plot. Drag and drop the following columns under the highlighted fields. RoomType will be located within the Dim_Reservations table and %Profit Margin, %Repeatguest, and AvgReservation/Month will be located within the Key Measures once expanded. Refer to the image below indicated with the highlighted parts and the circled tables.


After inserting these columns into their fields, the image below is how the scatter plot should look like so far.
